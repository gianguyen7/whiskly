### Decisions

- **Redesign before wiring** — user explicitly chose to do visual redesign pass first, then Sprint 1 feature wiring.
- **Landing page redesign** — logo mark (SVG), wordmark in Nunito Sans 600, slogan, single primary CTA ("Get Started") with inline "Already have an account? Sign in" link.
- **Logo as back-to-home link** — on /signup and /login pages, tapping the logo navigates to landing page.
- **"For You" as home after login** — middleware redirect changed from /catalog to /for-you. New nav order: For You, Catalog, Log, Discover, Profile.
- **Welcome modal on login** — slides up from bottom with random matcha fact, "Let's Go" CTA, session-scoped (shows once per browser session via sessionStorage).
- **Removed "See all" link from For You** — redundant with "Explore" button in hero. Will add back when recommendation engine has its own view.
- **Catalog filter system** — expandable filter panel with: Taste Notes (Umami, Sweet, Bitter, Grassy, Creamy), Good For (Latte, Usucha, Koicha, Cold Brew, Baking, Smoothie), Price Range (Budget/Mid/Premium/Luxury). Client-side filtering for now.
- **New matcha model fields** — added `price_per_gram`, `good_for`, `milk_pairing` to TypeScript model. DB migration not yet written.
- **Matcha card redesign** — shows taste highlights (auto-derived from scores >= 3.5), good-for tags, price/gram, milk pairing with droplet icon.
